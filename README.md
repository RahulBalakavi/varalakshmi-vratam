# Sri Varalakshmi Vratam — A Guided Course

A self-contained, narrated slide-deck that walks a devotee through the Sri Varalakshmi
Vratam step by step — the worship of the boon-granting Goddess invoked in the kalasham,
including the nine-knot toram rite — with each mantra in Devanagari + romanization, a
play-along reciter (Telugu audio + karaoke transcript), the Charumathi katha narrated in
Telugu with English read-along, and a semantic background image per phase.

**Live:** https://rahulbalakavi.github.io/varalakshmi-vratam/

Mantras are traditional liturgy (transcribed from public repositories: aupasana.com,
stotranidhi.com, drikpanchang.com); the 108 names are the Mahalakshmi Ashtottara. Story
narration is an original retelling of the traditional katha. Images are Creative-Commons /
public-domain from Wikimedia Commons, attributed on each slide. Voice: Sarvam AI.
